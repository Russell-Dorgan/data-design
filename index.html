REDDIT:

  • PROFILE:
  • profileId (primary key)
  • profileActivationToken (for account verification)
  • profilPhone (for verification requests)
  • profileEmail (for further authentication if needed)
  • profilePhone (for push notifications as well as potential password reset)
  • POST:
  • postID (to denote who started or contributed to the thread)
  • postProfileId (so users can go to users reddit profile and inspect what sort of threads the user is aprticpating in)
  • postTitle (to show name of thread)
  • postDateTime (to document when thread and or post was created)
  • postContent (area where text will go for comment or thread)
  • postImage (field so user can post html link to post an image)
  • INTERACTION:
  • interactionId (so users can see who posted what and navigate to their profile)
  • interactionPostId (so users can see the name of the member who posted)
  • interactionProfileId (to denote what other users commented on post)
  • interactionUpvote (BOOLEAN for upvoting or downvoting)
  • interactionDownvote (BOOLEAN for upvoting or downvoting)
  • interactionReplyId (so users can see who posts which replies)
  • interactionReply (so users can reply)
  • RELATION:
  • One reddit account can make multiple posts and or comments (1 to n)
  • Many forum post interactions can be tracked back to the users profile. (m to n)
