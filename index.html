REDDIT:

  • PROFILE:
  • profileId (primary key)
  • profileActivationToken (for account verification)
  • profileHandle (to denote your username)
  • profileEmail (for push notifications as well as potential password reset)
  • profilePhone (for further authentication if needed)
  • FORUM POST INTERACTIONS:
  • postID (to denote who started or contributed to the thread)
  • redditProfileId (so users can go to users reddit profile and inspect what sort of threads the user is aprticpating in)
  • threadPostDate (to document when thread and or post was created)
  • "LIKING" SYSTEM:
  • redditUpVote (so users can see if user upvoted or not)
  • redditDownVote (so users can see if user downvoted or not)
  • RELATION:
  • One reddit account can make multiple posts and or comments (1 to n)
  • Many forum post interactions can be tracked back to the users profile. (m to n)
